## How to install
```console
git clone https://github.com/lezetho/ticket-bot.git
cd ticket-bot
npm install
node .
```
## How to setup

Edit the `config.json`

```json
{
    "token": "YOUR_TOKEN_HERE",
    "prefix": "- (this is a example, you can do anything)",
    "serverName": "YOUR_SERVER_NAME_HERE"
}
```
